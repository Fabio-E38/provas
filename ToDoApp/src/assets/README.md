# Assets Organization

Questa cartella contiene tutti i file statici dell'applicazione.

## 📂 Struttura

- **images/** - Immagini e grafiche
  - **logos/** - Loghi aziendali
  - **icons/** - Icone custom (non da librerie)
  - **backgrounds/** - Immagini di sfondo

- **fonts/** - Font personalizzati (se non usi Google Fonts)

- **data/** - File JSON statici
  - Mock data per sviluppo
  - Configurazioni statiche

- **styles/** - File CSS/SCSS condivisi
  - Variabili CSS
  - Mixins
  - Temi

- **favicon.ico** - Icona del sito (mostrata nel browser tab)
